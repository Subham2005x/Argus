About Our Project

In today's fast-paced world, traditional attendance systems are outdated, time-consuming, and prone to errors. Our Automatic Attendance System using Facial Recognition offers a contactless, efficient, and secure solution for educational institutions, workplaces, and events.

This system utilizes ESP32-CAM to capture real-time images, which are then processed using advanced machine learning and computer vision techniques like OpenCV and dlib. The system ensures accurate face recognition, automatically marking attendance in a secure SQL database, eliminating the need for manual input.

Key Features

  1. Contactless & Secure – Eliminates manual attendance, reducing errors and proxy attendance.
  2. Fast & Real-Time Processing – Uses machine learning for instant face recognition.
  3. Low-Cost & Open-Source – No expensive commercial software or cloud fees required.
  4. Local Data Storage & Privacy – Attendance data is stored securely in SQLite/MySQL, ensuring privacy.
  5. Customizable & Scalable – Can be modified for schools, offices, and industries.

How It Works

  1. Image Capture: When a person enters, the ESP32-CAM module captures an image of their face and sends it to a Flask-based local server for processing.
  2. Face Detection & Preprocessing: The system detects faces in the captured image using OpenCV and MediaPipe. The face is then aligned, cropped, and converted into a numerical format for better recognition accuracy.
  3. Face Recognition & Authentication: The processed face is compared against stored face embeddings in the database using dlib's deep learning model and OpenCV . If a match is found, the person's identity is confirmed.
  4. Attendance Marking: Once authenticated, the system automatically logs the attendance into an SQLite database, ensuring accurate record-keeping.
  5. Dashboard & Analytics: A web-based dashboard allows administrators to view attendance records, generate reports, and monitor statistics in real-time with proper login system.

Unlike expensive commercial models, our system is affordable, open-source, and privacy-focused, making it ideal for institutions looking for a cost-effective yet powerful solution.
